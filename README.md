# SimpleAlbumList
Simple List Detail [Album -> Photos -> Photo Detail]
- Album List
- Photo List
- Photo Detail

Fetch data from https://jsonplaceholder.typicode.com

Load data using retrofit from fetch and livedata() as Observer
